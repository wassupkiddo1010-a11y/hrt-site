# HRT.org Design System (MASTER)

Generated with [UI UX Pro Max](https://github.com/nextlevelbuilder/ui-ux-pro-max-skill) — adapted for client brand.

## Pattern
**Trust & Authority + Social Proof** — healthcare telehealth landing page  
Hero → dual pathways (men/women) → trust strip → symptoms → treatments → process → testimonials → CTA

## Style
**Accessible & Ethical** + **Swiss Modernism** — clean white, subtle borders, no gradient blobs, WCAG AA contrast

## Typography
- **Headings:** Playfair Display — editorial, premium
- **Body:** Inter — clean, accessible, 16px base

## Anti-patterns (avoid)
- AI purple/pink gradients
- Neon colors
- Motion-heavy animations
- Emoji as icons
- Generic SaaS card overload

## Pre-delivery checklist
- [x] SVG icons only (Lucide)
- [x] cursor-pointer + hover 200ms on interactive
- [x] Focus rings visible
- [x] prefers-reduced-motion
- [x] Responsive 375 / 768 / 1024 / 1440
